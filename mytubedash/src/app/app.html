# Instalare Home Assistant Add-on

Această aplicație este optimizată pentru a fi servită dintr-un container via un Add-on custom în mediul Home Assistant (NUC).

terminal Pașii de instalare:

  1. Deschideți Home Assistant și navigați la Settings > Add-ons > Add-on Store.
  2. Apăsați pe iconița de meniu (3 puncte dreapta-sus) și alegeți Repositories.
  3. Adăugați URL-ul depozitului GitHub deținut de deținătorul proiectului:
    https://github.com/vlad2939/mytubedash-addon
  4. Reîncărcați lista și căutați "MyTube Dash" în panoul de add-on-uri.
  5. Apăsați Install. După instalare, bifați opțiunile "Show in sidebar".
  6. Porniți Add-on-ul (Start). Interfața web a aplicației (SPA) va fi accesibilă din meniul principal (Sidebar) sub formă de iFrame proxy.

# Despre "MyTube Dash"

Aceasta este o soluție simplistă dezvoltată pentru a viziona clipuri YouTube strict pe baza listelor personale, eliminând complet algoritmul de recomandare și elementele care distrag atenția.
